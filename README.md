**WEB RECON** is a powerful Python-based security tool designed for the initial reconnaissance phase of a penetration test. 
It automates the process of gathering critical data about a target website's infrastructure, server location, and security posture.

---

## üöÄ Features
* **Target IP Discovery:** Instantly resolves a website URL into its physical IP address.
* **IP Geolocation Tracking:** Identifies the target server's physical location, including Country, Region, City, and Zip Code.
* **Infrastructure Analysis:** Detects the Internet Service Provider (ISP) and Organization hosting the website.
* **Server Fingerprinting:** Reveals the web server software (e.g., Nginx, Apache, Cloudflare) and underlying technology stacks.
* **Security Audit:** Checks for the presence or absence of critical HTTP security headers.
